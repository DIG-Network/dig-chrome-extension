DIG Control Panel

Run the full DIG Network on your machine

You're browsing verified chia:// content through the hosted network right now. Install the DIG node to resolve it locally — faster, private, and available offline once cached — and to host and manage your own stores on the decentralized network.

Download the DIG node Learn how it works

No node? No problem — the extension keeps resolving chia:// content through the hosted network (rpc.dig.net). A local node just makes it yours.

⚡

Local & fast

Content resolves on your own machine instead of a remote gateway — lower latency and no round-trip to a shared server.

🔒

Private by default

Your reads stay on your device. Nothing about what you browse leaves your machine to a hosted endpoint.

📡

Host & earn

Run a node to host stores, serve the network, and take part in the decentralized DIG Network — not just consume it.

✈

Works offline

Once content is cached locally, it keeps resolving even without a connection to the hosted network.

Get set up in three steps

  1. Download the installer for Windows, macOS, or Linux — it installs the DIG node as a background service in one step.
  2. Let it register dig.local — the extension detects your node automatically and switches the resolver to it.
  3. Open this Control Panel from the extension to manage hosted stores, the local cache, and network sync.
