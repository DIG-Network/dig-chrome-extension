DIG Network settings

Configure how this browser resolves the DIG Network.

Appearance

Choose the theme for the DIG wallet (popup + full screen) and this settings page. New installs use the light theme; pick Dark, or System to follow your operating system.

Page toolbar

Show a slim DIG toolbar at the top of every page — one-click access to your Wallet, DIG Shields, and the Control Panel, plus "Verified on Chia" / "Loaded from local" badges when you are viewing DIG Network content. Off by default.

dig-node (local resolver)

Run the dig-node app to resolve chia:// content locally (the standalone twin of the native browser's in-process dig-node). Point this at the dig-node's address; leave it on the default to use the hosted RPC instead.

DIG RPC endpoint (upstream)
