DIG Network settings

Configure how this browser resolves the DIG Network.

Local content cache

Verified chia:// content this extension has fetched is cached in memory for this session to speed up repeat loads. It clears when the browser restarts.

dig-node (local resolver)

Run the dig-node app to resolve chia:// content locally (the standalone twin of the native browser's in-process dig-node). Point this at the dig-node's address; leave it on the default to use the hosted RPC instead.

DIG RPC endpoint (upstream)

Used to fetch ciphertext + Merkle proofs when no dig-node is configured. Verification and decryption always happen on this device.

Wallet (WalletConnect → Sage)

The extension connects your Chia wallet over WalletConnect to Sage. Enter a WalletConnect / Reown project id to enable pairing.
