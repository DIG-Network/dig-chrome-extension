# Chrome Web Store Listing Assets

This directory contains the official Chrome Web Store listing images for the DIG Network Browser Extension.

## Assets

- **store-icon-128.png** — 128×128, app icon for the store
- **screenshot-1..5-1280x800.png** — Five 1280×800 screenshots showing extension features
- **promo-small-440x280.png** — 440×280 small promotional tile
- **promo-marquee-1400x560.png** — 1400×560 marquee promotional image

All images are 24-bit PNG without alpha channel, matching Chrome Web Store specifications.

## Regeneration

These assets are generated from the repository's e2e screenshot corpus (`e2e/__screenshots__/`) and the DIG brand tokens. To refresh them, run the screenshot generation workflow and update this directory accordingly.
